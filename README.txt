Task ID: Survey_App
Framework : Django
This web application conducts a survey asking a few questions like name,state,city and college.
There is dynamic loading for state-cities.
Once a user successfully logs in he is allowed to fill the survey form.
There are various pages like index page which contains names of all participants of survey,once you click on one of those names you are taken to the page of their detailes.
There is also a page displaying the statics of the survey.
